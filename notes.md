# Proyecto 01 — Rock Paper Scissors

## Qué construí
Juego funcional de Piedra, Papel o Tijera contra la máquina
con lógica de repetición hasta que el usuario gane.

## Conceptos practicados
- Funciones con `def`
- Bucles `while` con condición de salida `break`
- Condicionales `if / elif / else`
- Listas y selección aleatoria con `random.choice()`
- f-strings para mostrar resultados dinámicos

## Dificultades encontradas
- Entender cuándo usar `break` para salir del bucle
- La indentación en Python es crítica — un espacio mal puesto rompe todo
